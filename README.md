# Untitled - Simple Launcher

## 프로그램 정보 (ver 1.0.0.0)

* Aki.Server와 Aki.Launcher의 기능을 하나의 프로그램만 실행해도 가능하게 만들었습니다.

* 단 계정을 생성하는 기능은 없기때문에 최초 설치시 Aki.Launcher로 계정을 만들어주셔야 합니다.

---

Config

`user\launcher\UntitledConfig.json`
